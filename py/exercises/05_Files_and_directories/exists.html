Does a file exist?

Prompt the user for the name of a file and then inform the user if the file exists or not. Tips: Methods to interact with the filesystem can be found in the os module, so start your program with the following line: import os The os.path.exists(path) method returns True if the specified path exists.

Sample output

Enter the path of a file: foo.txt

File not found: foo.txt


Enter the path of a file: exists.py

File exists: exists.py
