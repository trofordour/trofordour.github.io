Write a file

Make a new version of your times table program that writes its output to a .txt file. Tips Opening a file as shown below will allow you to call the .writelines() method on the file, to write out one or more lines. You must place the newline character ("\n") at the end of each string you write to ensure that it appears on its own line in the output. 
with open('hello.txt', 'w') as file:
    file.writelines("hello world\n")
