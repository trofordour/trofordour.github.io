Indexing the elements of a list

Write a program that starts with the following line:

colours = ['red', 'blue', 'green', 'brown', 'pink', 'yellow', 'purple']

By using the index operator, print the first, middle and last items of the colours list.


Sample output

Here is a list of colours: ['red', 'blue', 'green', 'brown', 'pink', 'yellow', 'purple']
The first element of the list is:  red
The last element of the list is:  purple
The middle element of the list is:  brown
