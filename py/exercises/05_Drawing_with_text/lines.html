Tips: The previous exercise showed how a canvas data structure (which is a two dimensional list of lists) can be used to plot pictures made of text characters. The function below can be used to plot a horizontal straight line on the canvas. 
def draw_line_horizontal(canvas, x1, y1, length, char):
    # Draw a horizontal line from (x1, y1) of the given length (in terms of number of characters)
    # This code assumes that all points on the line are within the boundaries of the canvas
    for y in range(y1, y1 + length):
        canvas[x1][y] = char
Here's how the above code can be used. 
canvas = create_canvas(5, 10, ".")

draw_line_horizontal(canvas, 1, 1, 8, "-")

display_canvas(canvas)
