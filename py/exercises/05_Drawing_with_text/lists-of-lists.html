Nested lists

By now you are familar with lists that contain simple types like numbers or strings, but what about a list that contains other lists? Here's an example: 
>>> items = [[1,2,3],[4,5,6],[7,8,9]]
>>> items
[[1, 2, 3], [4, 5, 6], [7, 8, 9]]
>>> items[0]
[1, 2, 3]
>>> items[1]
[4, 5, 6]
>>> items[2]
[7, 8, 9]
>>>
In the example above, each element in the list called items is another list. We refer to the inner lists as sub-lists or nested-lists. We can index the individual elements in each sub-list by introducing a second instance of the indexing operator. 
>>> items[0][0]
1
>>>
In the example above, the first use of [0] means get the first list. The second use of [0] means get the first element of the list obtained from the first use of [0]. In this fashion, all the elements of the original list can be referenced. 
>>> items[0][1]
2
>>> items[0][2]
3
>>> items[1][0]
4
>>> items[1][1]
5
>>> items[1][2]
6
>>> items[2][0]
7
>>> items[2][1]
8
>>> items[2][2]
9
>>>
It is usually impractical to deal with the elements of a large data structure like this each individual element at a time, so iteration is often used in conjunction with nested lists. 
>>> for x in range(3): 
...     for y in range(3): 
...             print(f"items[{x}][{y}] has value {items[x][y]}")     
...
items[0][0] has value 1
items[0][1] has value 2
items[0][2] has value 3
items[1][0] has value 4
items[1][1] has value 5
items[1][2] has value 6
items[2][0] has value 7
items[2][1] has value 8
items[2][2] has value 9
>>>
Below is a program that generates a list of lists to represent times tables from 1 to 12. 
times_tables = []

for i in range(1, 13):
    one_table = [i * j for j in range(1, 13)]

    times_tables.append(one_table)

print(times_tables)
Below is the output it produces.
