Making a canvas

In the previous exercise we used a list of lists to create a two dimensional grid, which we printed to create a crude diagram on the screen. This exercise takes these ideas a little further. The code below creates and displays a "canvas" of arbitrary length and width. The "." character is placed in every cell in the canvas so we can easily verify its dimensions. 
def create_canvas(x, y, char):
    canvas = []

    for i in range(x):
        row = [char for j in range(y)]

        canvas.append(row)

    return canvas

def display_canvas(canvas):
    for row in canvas:
        print("".join(row))

canvas = create_canvas(10, 25, ".")

display_canvas(canvas)

Aside: the usual convention in maths, which we learn in primary school, is that the x-axis is horizontal, and the y-axis is vertical. However, due to the way in which these pictures are being printed in text, our graphs are rotated, clockwise, by 90 degrees: the x-axis has become the vertical axis, and the y-axis as become the horizontal axis. While this isn't the way most of us have been trained to think about coordinates, it does not break anything. Our coordinate system is perfectly valid and consistent within itself, and the more you work with it the more you will get used to it. We can "draw" on the canvas simply by assigning single characters to the cells in the two dimensional list of lists. For example, the functions below can be used to create horizontal and vertical lines. 
def draw_horizontal(canvas, x, y1, y2, char):
    # Draw a line from (x, y1) to (x, y2)
    # This code assumes that all points are within the boundaries of the canvas and that y1 < y2
    for y in range(y1, y2):
        canvas[x][y] = char

def draw_vertical(canvas, x1, x2, y, char):
    # Draw a line from (x1, y) to (x2, y)
    # This code assumes that all points are within the boundaries of the canvas and that x1 < x2
    for x in range(x1, x2):
        canvas[x][y] = char
Here's how the above functions might be used. 
canvas = create_canvas(10, 25, ".")

draw_horizontal(canvas, 1, 2, 23, "_")

draw_vertical(canvas, 4, 8, 18, "|")

display_canvas(canvas)
