Password generator

Ask the user to enter a number, and then generate and print that number of random passwords. Each password should meet the following criteria: * Length: 16 characters. * One punctuation character. * A random number of upper-case letters, lower-case letters and digits, with at least one of each. For example, random.choice() can be used to randomly select an item from a list. >>> import random >>> random.choice(string.ascii_lowercase) 'v' The random.shuffle() method can be used to scramble the order of a list. Depending on your approach, you may want to use other random methods (no pun intended).
