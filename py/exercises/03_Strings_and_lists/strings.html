Working with strings

For this exercise, follow along with the examples given below in the Python REPL. Experiment with the sample code, to ensure you understand the concepts presented. A value enclosed in single or double quotes is called a string. In many situations, a string can be treated as a list of characters. For example, a string can be passed to the len() function, and the index operator (square brackets) can be used with strings: 
>>> 'hello world'
'hello world'
>>> len('hello world')
11
>>> 'hello world'[0]
'h'
>>>
We can iterate over the characters in a string like this: 
>>> for c in "hello world":
...     print(c)
... 
h
e
l
l
o

w
o
r
l
d
>>>
A string can be converted into a true list of characters using the list() function: 
>>> list('hello world')
['h', 'e', 'l', 'l', 'o', ' ', 'w', 'o', 'r', 'l', 'd']
>>>
A list of characters (or a list of strings) can be converted into a string using the .join() method: 
>>> list_of_chars = list('hello world')
>>> list_of_chars
['h', 'e', 'l', 'l', 'o', ' ', 'w', 'o', 'r', 'l', 'd']
>>> 
>>> "".join(list_of_chars)
'hello world'
>>> 
>>> "-X-".join(list_of_chars) 
'h-X-e-X-l-X-l-X-o-X- -X-w-X-o-X-r-X-l-X-d'
>>>
A string with nothing in it ('') is called an empty string. 
>>> ''
''
>>> len('')
0
>>> list('')
[]
>>>
As with lists, strings can be joined together to form new strings using the plus operator. This is called concatenation. 
>>> 'hello' + ' ' + 'world'
'hello world'
>>> greeting = 'hello'
>>> greeting
'hello'
>>> greeting += ' '
>>> greeting
'hello '
>>> greeting += 'world'
>>> greeting
'hello world'
>>>
One critical way in which a string differs from a list of characters is that a string is immutable, which means that, once a string has been created it cannot be changed: 
>>> 'hello world'[0] = 'H'
Traceback (most recent call last):
  File "", line 1, in 
TypeError: 'str' object does not support item assignment
>>>
>>> list_of_chars
['h', 'e', 'l', 'l', 'o', ' ', 'w', 'o', 'r', 'l', 'd']
>>> list_of_chars[0] = 'H'
>>> "".join(list_of_chars)
'Hello world'
>>>
It is therefore always necessary to create a new string in order to create a different string value. If a new string value needs to be based on an existing string value, then it is common to use indexing operators, or the .replace() method, to construct the new string: 
>>> 'hello world'.replace('h', 'H')
'Hello world'
>>> 'hello world'.replace('hello', 'goodbye')
'goodbye world'
>>> 'hello world'.replace('o', 'u')
'hellu wurld'
>>>
>>> greeting = 'hello world'
>>> greeting
'hello world'
>>> 'H' + greeting[1:]
'Hello world'
>>>
