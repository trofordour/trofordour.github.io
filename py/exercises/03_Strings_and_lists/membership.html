Testing for membership

For this exercise, follow along with the examples given below in the Python REPL. Experiment with the sample code, to ensure you understand the concepts presented. The in operator can be used to answer the question: does a sequence of items (like a list or a string, for example) contain a given sub-sequence? It returns simply True or False (i.e. it's a Boolean operator). Conversely, the not in operator returns True if a given sequence does not contain a given sub-sequence; and False otherwise. The following examples show how this works with strings. 
>>> 'o' in 'hello world'
True
>>> 'a' in 'hello world'
False
>>> 'world' in 'hello world'
True
>>> 'apple' not in 'hello world'
True
>>>
The following examples show how this works with lists. 
>>> fruit = ['apple', 'banana', 'kiwi', 'orange', 'pineapple']
>>> 'kiwi' in fruit
True
>>> ['kiwi', 'orange'] in fruit
False
>>> 'pear' in fruit
False
>>> 'pear' not in fruit
True
>>>
