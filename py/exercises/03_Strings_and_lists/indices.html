Indexing the elements of a list

Write a program that starts with the following line: 
colours = ['red', 'blue', 'green', 'brown', 'pink', 'yellow', 'purple']
By using the index operator (use square brackets), print the first, middle and last items of the colours list. Use slice notation (use a colon) to display the following elements of the list:
  • The second to fourth items.
  • Everything after the first item.
  • Everything before the last item.
Use the .index() method to display the index of 'yellow'.

Sample output

Here is a list of colours: ['red', 'blue', 'green', 'brown', 'pink', 'yellow', 'purple']

First element of the list:  red

Last element of the list:  purple

Middle element of the list:  brown

Second to fourth items:  ['blue', 'green', 'brown']

Everything after the first item:  ['blue', 'green', 'brown', 'pink', 'yellow', 'purple']

Everything before the last item:  ['red', 'blue', 'green', 'brown', 'pink', 'yellow']

Index of yellow:  5
