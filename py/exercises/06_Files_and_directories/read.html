Type the contents of a file

Prompt the user for the name of a file. Read and print the contents of the file. Tips: Use the with-statement in conjunction with the open(path, 'r') method to open a file for reading. For example, the following line opens the file, specified by the variable path, for reading: 
with open(path, 'r') as myfile:
Inside the with-block, the variable myfile can be used to perform operations on the file. For example, the .read() method reads the contents of a file: 
with open(path, 'r') as myfile:
    contents = myfile.read()
