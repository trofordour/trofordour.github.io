Display contents of a directory

Prompt user to enter the name of a folder. Print the contents of the folder. Prefix the name of each directory with "D -". Prefix the name of each file with "F -". Tips: The os.path.isdir(path) method returns True if path identifies a directory (folder). The os.listdir(path) method returns the contents of a folder, as a list of file and directory names. Remember: . means the current directory and .. means the parent of the current directory.

Sample output

Enter the path of a directory: foo

Directory not found: foo


Enter the path of a directory: .

Contents of directory: .

F - list-files.py
F - _description.txt
F - _example.txt


Enter the path of a directory: ..

Contents of directory: ..

D - exists
D - list-files
