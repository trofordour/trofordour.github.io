Draw

Write a program that allows the user to interactively draw text-based pictures.

Sample output

Available commands:
  canvas x y char - discard any existing canvas and create a new canvas with height x and width y
  help - print this help message
  hline x y length char - draw a horizontal line from point (x, y) with the given length
  vline x y length char - draw a veritical line from point (x, y) downwards, with the given length
  rectangle x y height width char - draw a rectangle with top-left corner at (x, y) with the given height and width
  save filename.txt - save the current canvas to the given filename
  load filename.txt - intialize the canvas from the contents of a file
  quit
