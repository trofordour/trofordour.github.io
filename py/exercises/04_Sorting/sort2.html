Function input parameters are a list of numbers and two indices (i and j). The function must compare the values in the list specified by the two indices.
  • If the first value is greater than the second value then the function should swap them.
  • Otherwise, the function should leave the list unchanged.
Use the function to sort the first two numbers of a random list. Tips: Use the swap() function from the previous exercise to swap list items.
