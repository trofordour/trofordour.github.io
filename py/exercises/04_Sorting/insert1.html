Insert one item into a sorted list

Create a new program that begins with the following line: 
nums = [4, 12, 15, 27, 29, 46, 61, 76, 94, 96]
nums is a sorted list of random numbers (sorted lowest to highest). Ask the user to enter a number. Convert the input value from a string to an int using the int() function. Insert the entered number into the list above, maintaining the sort order (lowest to highest). This requires iterating over the list, from beginning to end, comparing each existing list item with the new value, until either:
  • an existing item is found that is larger than the new value, which means that the new value should be inserted before this bigger item;
  • or, the end of the list is reached, which means the new value is bigger than any existing item, and the new value should be appended to the list.
