Insertion sort based on creating a new list

Implement insertion sort on a list of random numbers. To perform insertion sort, we iterate over the original list. We insert each item from the original list into a new sorted list, placing each item in the correct position in the new list. The new sorted list is initially empty i.e. it can be initialized as follows: 
sorted_list = []
Inserting an item into the correct position in a sorted list can be easily accomplished using the insert1 function we developed in the previous exercise.

Sample output

List length: 10

[74]

Inserting 49 into [74]

Inserting 100 into [49, 74]

[37, 41, 46, 49, 65, 70, 74, 100]
