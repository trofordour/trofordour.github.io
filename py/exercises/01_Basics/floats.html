Floating-point numbers

Computers represent real numbers using floating-point numbers (or floats for short). To say it another way: a floating-point number is a number with a decimal point. Write a simple program that prompts for the radius of a circle, and calculates the circle's circumference and area. Tips: Circumference = 2 * π * r Area = π * r ** 2 Use a constant value of 3.142 for π. The float() function can be used to convert a string to a float.

Sample output

Enter the radius of a circle: 1.0

Circumference of your circle = 6.284

Area of your circle = 3.142
