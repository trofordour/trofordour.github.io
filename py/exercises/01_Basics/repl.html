REPL - Read, Execute, Print, Loop

A program that prompts for a command, executes it, and then prompts for a another command in order to repeat the process, is often referred to as a REPL, which stands for:
  • Read command
  • Execute command
  • Print result
  • Loop back to the beginning
Every program you can install to work with Python provides a REPL that understands how to interpret and execute Python. One of the first things you should do is find out how to run the REPL for the version of Python that you have installed. The REPL provides an invaluable tool for understanding how Python works in practice. When you are stuck with something, use the REPL to experiment with different ideas and potential solutions.
