Types

Write a program that starts with the following function. 
def type_of(value):
    return value.__class__.__name__
The function returns the name of the type of the value it is given, as a string. Use the function to display the type names of various different values.

Sample output

Type of 'hello world' is str

Type of 1 is int

Type of 1.0 is float

Type of True is bool

Type of [] is list

Type of () is tuple

Type of {} is dict
