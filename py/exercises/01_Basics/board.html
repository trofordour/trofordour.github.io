Chess board

Print a rough 8x8 chess board made of 8x3 blocks of asterisk and space characters.

Tips:

Defining these ("constant") variables at the start of the program make this task relatively easy to complete.

BORDER = " ------------------------------------------------------------------ "

BLACK_SQUARE_FIRST = "| ********        ********        ********        ********         |"

WHITE_SQUARE_FIRST = "|         ********        ********        ********        ******** |"
