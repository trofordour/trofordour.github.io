Compare integers

Selection is a fancy computer science term for chosing between one of two possible courses of action based on evaluating some logical condition. Write a simple program that prompts for two integer values and then determines which one is larger. Tips: Use the input() function to prompt the user for a number. The input() function returns a string. Use the int() function to convert input strings into integers.

Sample output

Enter an integer: 15

Enter another integer: 19

15 is less than 19


Enter an integer: 19

Enter another integer: 15

19 is greater than 15


Enter an integer: 28

Enter another integer: 28

28 is equal to 28
