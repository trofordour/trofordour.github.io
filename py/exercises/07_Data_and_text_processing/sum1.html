Read list of numbers from a text file and sum them

Reading and displaying the contents of text files has been covered in previous exercises. Now we can learn how to start doing something useful with the data that we find in files, beyond simply displaying it. Write a program that prompts the user for the name of a file that contains a list of numbers, one number per line. The program should read the contents of the file, and then calculate and print the sum of the numbers. Tips: The following code will split a string of text that contains newline characters ('\n') into a list of strings, where each string in the list is the contents of one line. The newline characters are removed from the strings (which is usually what we want). 
with open(path, 'r') as file:
    contents = file.read()

lines = contents.splitlines()
The following list comprehension will convert a list of strings into a list of integers. 
numbers = [int(line) for line in lines]
