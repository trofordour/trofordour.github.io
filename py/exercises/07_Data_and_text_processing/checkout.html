Supermarket checkout

Given a file called products.txt that contains a list of the products stocked in a supermarket, with their prices, and a file called shopping.txt that contains a list of items that a shopper is purchasing, and their quantities, print the receipt, which includes a calculation of the overall cost to the shopper. Tips: The input data files (see example output below) contain structured data. Each line consists of two fields separated by a separator character.
  • In products.txt, there are two fields (product name and product price) seperated by the "$" character.
  • In shopping.txt, there are two fields (product name and quatity) separated by the "#" character.
Each line of data can be easily broken up into its component parts by using the .split() method. For example, the following code splits a string on the '$' character, and then extracts the product name and price, stripping the whitespace from each field, and converting its data type, if necessary: 
    tokens = product.split('$')

    product_name = tokens[0].strip()
    product_price = float(tokens[1].strip())
