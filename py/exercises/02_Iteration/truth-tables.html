Logical operators

Print the truth tables for logical-not, logical-and, and logical-or. Tips: The following code can be used to iterate over the range of all possible combinations of two truth values (aka Booleans). 
truth_values = [True, False]

for a in truth_values:
    for b in truth_values:
        # Do something with a and b...
Use the following, slightly cryptic, f-string syntax, to print a Boolean value (True or False) in a field of width 6 (to get columns of truth values that are neatly aligned): 
>>> x = True                       
>>> y = False 
>>> print(f"|{x!s:6}|-|{y!s:6}|")   
|True  |-|False |
>>>

 A     ¦ not A
---------------

Truth table for logical-and

Truth table for logical-or

 A     ¦ B     ¦ A or B
------------------------
