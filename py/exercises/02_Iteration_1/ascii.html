ASCII character set

Use the chr() function to print the ASCII characters from 32 to 126. Tips: If n is variable that contains a number, the number can be printed in a width of three spaces like this: print(f"{n:3}") This is handy when you want to nicely format a list of 1-, 2- and 3-digit numbers.
